Grammar Review: Used To (Past Habits) üìù

Remember, used to is a fixed structure‚Äîyou do not change the form of the main verb!

Structure: Used To

Form Structure Example
Positive (+) Subject + used to + Base Verb (V1) I used to live in Turkey.
Negative (-) Subject + never used to + Base Verb (V1)
(OR Subject + didn't use to)		 I never used to like spicy food.
Question (?) Did + Subject + use to + Base Verb (V1) Did you use to play soccer?

Verb Agreement Review (Simple Present)

Reminder on forming negatives in the Present Simple, which is essential for contrasting with past habits:

  • I, You, We, They: Use don't (e.g., I don't play basketball at all.)
  • He, She, It: Use doesn't (e.g., My manager doesn't think.)
